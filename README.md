## 🚀 Quick Start

<table>
<tr>
<td>

### Prerequisites

Before you begin, ensure you have:

- **Node.js** `>= 14.x` — [Download](https://nodejs.org/)
- **npm** or **yarn** package manager
- Basic knowledge of React

</td>
<td>

### Verify Installation

```bash
node --version
npm --version
```

</td>
</tr>
</table>

### Installation Steps

**1️⃣ Clone the repository**
```bash
git clone https://github.com/krishbhuva03/Krish-PortFolio.git
cd Krish-PortFolio
```

**2️⃣ Install dependencies**
```bash
npm install
# or
yarn install
```

**3️⃣ Start development server**
```bash
npm start
```
🎉 Opens automatically at `http://localhost:3000`

**4️⃣ Build for production**
```bash
npm run build
```
📦 Creates optimized build in `build/` folder

## 💡 Customization Guide

<details>
<summary><b>🎨 Changing Colors</b></summary>

Edit `src/style.css`:

```css
:root {
  --primary-color: #6A0DAD;
  --background-color: #0c0513;
  --text-color: #FFFFFF;
}
```

</details>

<details>
<summary><b>✏️ Updating Content</b></summary>

- **About**: Edit `src/components/About/About.js`
- **Projects**: Edit `src/components/Projects/Projects.js`
- **Skills**: Update tech stack in `src/components/About/Techstack.js`
- **Resume**: Replace `src/Assets/resume.pdf`

</details>

<details>
<summary><b>🖼️ Changing Images</b></summary>

Replace files in `src/Assets/`:
- Avatar: `avatar.svg`
- Projects: `Projects/your-image.png`
- Background: `home-bg.jpg`

</details>
